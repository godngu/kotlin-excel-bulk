# 대용량 엑셀 Writer

## 왜 이렇게 만들었는가?

- DB 조회되는 데이터가 많은 경우 `List`로 받는 경우 OOM 발생 함.
- Excel로 Write 해야 하는 데이터가 많아서 시간이 오래 걸림

## 해결 방안은?

### OOM (Out Of Memory)

- MyBatis의 `ResultHandler`를 사용 해도 되지만 `JDBCTemplate`를 사용 하기로 했다.
- JDBCTemplate 에서는 `queryForStream`을 제공한다 -> 스크림 처리 가능
